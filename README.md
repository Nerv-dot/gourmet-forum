# Форум гурманов 🍴

Учебный проект — простой веб-форум для любителей вкусно поесть и готовить.

## Возможности
- Просмотр разделов форума
- Чтение тем и рецептов
- Просмотр комментариев
- Форма добавления новой темы
- Профиль пользователя
- Адаптивный дизайн
- Модальное окно приветствия с выбором входа/гостевого режима

## Технологии
- HTML5
- CSS3 (адаптивная верстка, flexbox)
- JavaScript (модальное окно, переключение форм)

## Страницы
- Главная / Разделы форума
- Список тем
- Просмотр темы с комментариями
- Создание новой темы
- Профиль пользователя
- Вход / Регистрация

## Как запустить
Просто открой `index.html` в любом современном браузере.
